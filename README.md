# AegisOverhaul

Windows PC のメンテナンスとシステム・ネットワークの最適化を一括で実行する包括的なバッチスクリプトです。
不要なファイルの削除、アプリケーションの更新、システム設定の最適化、ネットワークの調整などを自動化し、PC のレスポンス向上と動作の安定化を図ります。

## 主な機能

### 1. アプリケーションの最新化
- **一括アップデート**: インストールされているアプリケーションを `winget` を通じて最新の状態に更新します。
- **Microsoft Store の修復**: ストアアプリのリセットと再登録を行い、不具合を解消します。

### 2. システムおよびブラウザの徹底クリーンアップ
- **キャッシュの削除**: システム、アプリ（Discord, Office, Teams等）、および主要なブラウザ（Chrome, Edge, Brave, Vivaldi等）の不要なキャッシュや一時ファイルを削除します。
- **不要データの除去**: ゴミ箱の消去、システムログ、イベントログ、および肥大化した一時ディレクトリをクリーンアップします。
- **Windows Update メンテナンス**: 過去の更新プログラムの残骸を整理し、ストレージ容量を確保します。

### 3. システムパフォーマンスの最適化
- **OS レスポンスの向上**: スタートアップ時のアプリ起動遅延の解消、UI の応答速度向上、電力スロットリングの無効化を行います。
- **ゲーム・マルチメディア最適化**: ゲームプロセスの優先度調整や、リモートデスクトップの 60FPS 化、GameDVR（録画機能）の最適化を実施します。
- **電源設定の適正化**: 電源プランを標準の状態にリセットし、スリープ・休止状態やカバーを閉じた際の挙動をデスクトップ/ノートPC向けに最適化します。
- **ストレージメンテナンス**: SSD に対して TRIM コマンドを実行し、パフォーマンスの低下を防止します。

### 4. ハードウェアと描画の安定化
- **オーディオの安定化**: 音声の途切れやノイズを防止するため、USB の電力管理設定やマルチメディアタスクの優先度を調整します。
- **グラフィックスの最適化**: NVIDIA GPU の割り込み処理（MSI）を最適化し、MPO（マルチプレーンオーバーレイ）を無効化することで、画面のちらつきやスタッターを低減します。

### 5. ネットワークの高速化・安定化
- **プロトコルの最適化**: ネットワーク設定（TCP/IP）のリセットと、最新の輻輳制御プロトコル（BBR2 等）への最適化を行います。
- **DNS の最適化**: 応答性が高く安定した DNS サーバー（Google Public DNS 等）に設定し、キャッシュをクリアして接続トラブルを解消します。

### 6. その他のメンテナンス
- **セキュリティの更新**: Windows Defender の定義ファイルを最新の状態に更新します。
- **システム時計の同期**: 日本国内の信頼性の高い NTP サーバー（ntp.jst.mfeed.ad.jp）と同期し、時刻のズレを修正します。
- **エクスプローラーの修復**: フォルダ名が英語表記になってしまう問題の修復や、フォルダ表示設定の初期化を行います。

## 動作環境

- **OS**: Windows 10 / 11
- **権限**: 管理者権限（「管理者として実行」が必須）
- **インターネット接続**: アプリの更新や時刻同期、Defender の更新に必要です。

## 使い方

1. `AegisOverhaul.bat` を右クリックします。
2. 「**管理者として実行**」を選択します。
3. 黒い画面（コマンドプロンプト）が表示され、自動的に各項目の処理が開始されます。
4. 処理中にエクスプローラーが一時的に再起動されますが、これは正常な動作です。
5. 「続行するには何かキーを押してください...」と表示されたら完了です。設定を完全に反映させるため、PC の再起動を推奨します。

## 注意事項

- **自己責任で実行してください**: システムの深い部分の設定変更やファイルの削除を行うため、実行前に重要なデータのバックアップを推奨します。
- **環境への影響**: ネットワークのリセットにより、実行中に一時的に通信が切断されます。また、一部のアプリのログイン状態がクリアされたり、ブラウザのキャッシュが削除されることがあります。

## カスタマイズ

特定の処理を無効にしたい場合は、`AegisOverhaul.bat` をテキストエディタで開き、該当するコマンドの先頭に `rem ` を追記してコメントアウトしてください。
