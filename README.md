# AegisOverhaul

Windows PC のメンテナンスとネットワーク最適化を包括的に実行するバッチスクリプトです。
アプリの更新、キャッシュの削除、システム設定の最適化、ネットワークの調整などを自動化し、PC のパフォーマンスと安定性を向上させます。

## 主な機能

### 1. アプリケーションの最新化
- **一括アップデート**: `winget` を使用して、インストール済みのすべてのアプリケーションを最新バージョンに更新します。
- **Store アプリの修復**: Microsoft Store アプリの再登録とリセットを行い、動作を安定化させます。

### 2. システムおよびブラウザのクリーンアップ
- **キャッシュ削除**: Windows システム、Discord、Microsoft Office などの一時ファイルやキャッシュを削除します。
- **ブラウザの最適化**: Chrome, Edge, Brave, Vivaldi などの主要ブラウザのキャッシュ、サービスワーカー、IndexedDB などをクリーンアップします。
- **不要ファイルの除去**: ゴミ箱の消去、システムログの削除、および肥大化した一時ディレクトリのクリーンアップを行います。

### 3. システムのパフォーマンス向上
- **レスポンス最適化**: UI の応答速度向上、スタートアップ時のアプリ起動遅延の解消、および電力スロットリングの無効化を行います。
- **ゲーム・マルチメディア最適化**: ゲームモードの優先度調整や、リモートデスクトップの 60FPS 化などを実施します。
- **ストレージメンテナンス**: SSD に対して TRIM コマンドを実行し、書き込みパフォーマンスを維持します。

### 4. ハードウェアと描画の安定化
- **オーディオ最適化**: 音声の途切れを防止するため、USB の省電力設定の調整やマルチメディアタスクの優先度向上を行います。
- **グラフィックス最適化**: NVIDIA GPU の割り込み処理を最適化し、MPO（マルチプレーンオーバーレイ）を無効化することで、画面のちらつきやスタッターを低減します。

### 5. ネットワークの高速化・安定化
- **DNS と TCP の最適化**: DNS キャッシュのクリア、Winsock のリセット、および最新のネットワークプロトコル（BBR2 等）への最適化を行います。
- **DNS サーバー設定**: 応答性と安定性に優れた Google Public DNS（8.8.8.8 / 8.8.4.4）を優先的に使用するように設定します。

### 6. その他のメンテナンス
- **セキュリティ更新**: Windows Defender の定義ファイルを最新の状態に更新します。
- **時刻同期**: 日本国内の信頼性の高い NTP サーバーと同期し、システム時計を正確に保ちます。
- **システム修復**: 英語表記になってしまったシステムフォルダ名の修復などを実行します。

## 動作環境

- **OS**: Windows 10 / 11
- **権限**: 管理者権限（「管理者として実行」が必須）
- **前提条件**: `winget` および `PowerShell` が利用可能であること

## 使い方

1. `AegisOverhaul.bat` を右クリックします。
2. 「**管理者として実行**」を選択します。
3. 黒い画面（コマンドプロンプト）が表示され、処理が開始されます。
4. 処理中にエクスプローラーが一時的に再起動されますが、これは正常な動作です。
5. 「続行するには何かキーを押してください...」と表示されたら完了です。必要に応じて PC を再起動してください。

## 注意事項

- **自己責任で実行してください**: システム設定の変更やファイルの削除を伴うため、実行前に重要なデータのバックアップを推奨します。
- **環境への影響**: ネットワークのリセットにより、一時的に通信が切断される場合があります。また、一部のアプリのログイン状態やキャッシュがクリアされることがあります。

## カスタマイズ

特定の処理を無効にしたい場合は、`AegisOverhaul.bat` をメモ帳などで開き、該当する行の先頭に `rem ` を入力してコメントアウトしてください。
