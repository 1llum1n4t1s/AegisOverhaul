# AegisOverhaul

Windows PC のメンテナンスとシステム・ネットワークの最適化を一括で実行する包括的なバッチスクリプトです。
不要なファイルの削除、アプリケーションの更新、システム設定の最適化、ネットワークの調整などを自動化し、PC のレスポンス向上と動作の安定化を図ります。

## 主な機能

### 1. アプリケーションの最新化
- **一括アップデート**: インストールされているアプリケーションを最新の状態に更新します。
- **Microsoft Store の修復**: ストアアプリのリセットと再登録を行い、動作を安定化させます。

### 2. システムおよびブラウザの徹底クリーンアップ
- **キャッシュの削除**: システム、アプリ（Discord, Office, Teams等）、および主要なブラウザの不要なキャッシュや一時ファイルを削除します。
- **不要データの除去**: ゴミ箱、システムログ、イベントログ、および肥大化した一時ディレクトリをクリーンアップします。
- **Windows Update メンテナンス**: 過去の更新プログラムの残骸を整理し、ストレージ容量を確保します。

### 3. システムパフォーマンスの最適化
- **OS レスポンスの向上**: 起動時の不要な待機時間の排除、バックグラウンドでの予測読み込み負荷の軽減、UI の応答速度向上、電力効率の最適化を行います。
- **ゲーム・マルチメディア最適化**: ゲームプロセスの優先度調整や、画面録画機能の最適化を実施します。
- **電源設定の適正化**: 電源プランを標準状態にリセットし、スリープやカバーを閉じた際の挙動を使いやすく調整します。
- **ストレージメンテナンス**: SSD のパフォーマンス維持に必要なメンテナンス（TRIM）を実行します。

### 4. ハードウェアと描画の安定化
- **オーディオの安定化**: 音声の途切れやノイズを防止するため、電力管理設定や処理の優先度を調整します。
- **グラフィックスの最適化**: 画面のちらつきやスタッター（カクつき）を低減し、表示の安定性を向上させます。

### 5. ネットワークの高速化・安定化
- **設定のリセットと最適化**: ネットワーク設定をクリーンな状態にリセットし、通信効率を向上させるプロトコル設定を適用します。
- **接続トラブルの解消**: ネットワークキャッシュをクリアし、接続の安定性を高めます。

### 6. その他のメンテナンス
- **セキュリティの更新**: Windows Defender の定義ファイルを最新の状態に更新します。
- **システム時計の同期**: 信頼性の高い日本の時刻サーバーと同期し、時刻のズレを修正します。
- **エクスプローラーの修復**: フォルダ名が英語表記になる問題の修復や、表示設定の初期化を行います。

## 動作環境

- **OS**: Windows 10 / 11
- **権限**: 管理者権限（「管理者として実行」が必須）
- **インターネット接続**: 各種アップデートや時刻同期に必要です。

## 使い方

1. `AegisOverhaul.bat` を右クリックして「**管理者として実行**」を選択します。
2. 自動的に処理が開始されます。処理中にエクスプローラーが一時的に再起動されますが、正常な動作です。
3. 完了メッセージが表示されたら、設定を完全に反映させるため PC を再起動してください。

## 注意事項

- **自己責任で実行してください**: システム設定の変更やファイルの削除を伴うため、実行前にバックアップを推奨します。
- **環境への影響**: 実行中に一時的に通信が切断されたり、一部のアプリのログイン状態がクリアされることがあります。

## カスタマイズ

特定の処理を無効にしたい場合は、`AegisOverhaul.bat` をテキストエディタで開き、該当する行の先頭に `rem ` を追記して保存してください。
